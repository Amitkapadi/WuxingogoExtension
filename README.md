# WuxingogoExtension

## Intro
------

My Unity Editor extension.


##Features
------

* Export UGUI.
* Virtual variable edit.
* Create AssetBundle extension (Unity version 4.x).
* Quick set unity prefs.
* The Editor Window of generate C# code(CodeDom).
* Finite-State-Machine
